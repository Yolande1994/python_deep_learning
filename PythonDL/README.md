# PythonDL 深度学习基础与实战项目集

本仓库为深度学习算法学习与实战项目集合，覆盖经典机器学习分类器、卷积神经网络、迁移学习等核心内容，包含原理级手动实现与工程化实现两种形态。

## 目录结构

### classifiers/  经典机器学习分类器（NumPy手动实现）
基于NumPy从零实现主流分类算法，包含完整的损失函数推导与梯度反向传播，用于理解算法底层原理：
- K近邻（KNN）：支持双循环、单循环、纯向量化三种L2距离实现
- 线性分类器：SVM合页损失、Softmax交叉熵损失
- 两层全连接神经网络：手动实现前向传播与反向传播

### code/  深度学习工程级项目
完整可运行的深度学习项目：
- CIFAR-10 图像分类：基于PyTorch的轻量级CNN，包含数据增强、训练验证、结果可视化全流程

### cat_dog_classify/  迁移学习猫狗二分类项目
基于ResNet101预训练模型的图像分类实战项目：
- 数据集自动划分脚本（训练集/验证集8:2随机划分）
- 迁移学习全参数微调训练
- 学习率阶梯衰减、指标打印、模型权重保存

### datasets/  数据集资源与加载工具
- CIFAR-10 数据集自动下载脚本
- 数据加载与预处理工具函数

### tutorials/  入门练习与算法演示
深度学习基础语法与算法验证Notebook，用于入门学习与原理演示：
- PyTorch张量基础操作
- 线性回归、感知机、异或问题入门实现
- KNN、SVM、Softmax、两层神经网络算法演示

### utils/  通用工具集
- 数值梯度校验工具，用于验证反向传播梯度正确性
- 图像、距离矩阵等可视化工具函数

## 环境依赖
- Python 3.8+
- NumPy
- PyTorch 2.0+
- torchvision
- Matplotlib
