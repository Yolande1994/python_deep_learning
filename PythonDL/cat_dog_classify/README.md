# 猫狗图像分类（迁移学习）

基于ResNet101预训练模型的二分类迁移学习项目，在猫狗数据集上完成全参数微调。

## 功能特性
- 数据集自动划分：按8:2比例随机划分训练集与验证集，保留文件元信息
- 迁移学习训练：加载ImageNet预训练权重，替换分类头进行全参数微调
- 训练流程：支持学习率阶梯衰减、训练/验证指标实时打印、模型权重自动保存

## 文件说明
- `config.py`：训练超参数配置
- `data_split.py`：数据集划分脚本
- `net.py`：ResNet101分类网络定义
- `train.py`：训练主程序，包含训练+验证完整流程

## 使用方式
1. 将原始猫狗图片按 `cat/`、`dog/` 文件夹放入 `datasets/` 目录
2. 执行 `python data_split.py` 完成训练集/验证集划分
3. 修改 `config.py` 调整批次大小、学习率、训练轮数等参数
4. 执行 `python train.py` 启动训练，训练完成后权重保存在 `save_dir` 目录
