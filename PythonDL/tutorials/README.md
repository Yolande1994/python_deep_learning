# 入门练习与算法演示

深度学习基础语法与算法验证Notebook集合，用于入门学习与原理演示。

## 包含内容
### 基础语法练习
- `torch_tensor_operations.ipynb`：PyTorch张量基础操作、广播机制、设备迁移
- `LinearRegression.ipynb`：NumPy手动实现线性回归 + PyTorch框架实现对比
- `preceptor.py`：感知机算法实现
- `xor.py`：异或问题神经网络实现
- `fc.py`：全连接层基础实现

### 算法演示Notebook
- `knn.ipynb`：K近邻算法原理验证与效果演示
- `svm.ipynb`：SVM合页损失、梯度计算原理验证
- `softmax.ipynb`：Softmax交叉熵损失原理验证
- `two_layer_net.ipynb`：两层神经网络前向/反向传播原理演示
